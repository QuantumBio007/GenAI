# GenAI Homework 1 – Coding Assistant Onboarding

This repository contains my Homework 1 for the GenAI course, where I used an AI coding assistant to create and edit markdown files and practiced a basic Git/GitHub workflow.

## Git workflow used

I worked from a local folder linked to this GitHub repository. I created the markdown files (`README.md`, `assistant_draft.md`, `final_version.md`), used my coding assistant to generate the initial draft, then made manual edits and committed changes along the way. The main Git commands I used were: `git init`, `git add`, `git commit`, `git status`, `git pull`, and `git push`.

## Commit summary

- `Initial repository setup` – created the repository and initial markdown files.  
- `Add assistant-generated draft` – used the coding assistant to populate `assistant_draft.md` and documented the prompt and reflections.  
- `Add manually edited final version` – copied the draft into `final_version.md`, made my own edits, and documented what I changed and why.  
- `Update README with workflow details` – added this description of the repository and Git steps.
